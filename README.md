🚀 Roadmap Tracker (In Development)
----------------------------------------------------

Roadmap Tracker is a career learning tracker that helps users follow structured roadmaps for different tech careers and track their progress step-by-step.
This project is currently under development and new features are being added regularly.
🎯 Project Goal
The goal of this project is to build a dashboard-style roadmap tracker where users can:
Select a career path
Follow a structured roadmap
Track completed learning steps
Monitor their learning progress visually

----------------------------------------------------
✨ Features Implemented So Far
✔ Career Roadmap Generator
✔ Interactive Task Checklist
✔ Live Progress Counter
✔ Progress Bar Visualization
✔ LocalStorage for saving progress
✔ Separate progress tracking for each career
✔ Sidebar Dashboard Layout
✔ Component-based architecture using Svelte
